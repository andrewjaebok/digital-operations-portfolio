# Digital Operations Portfolio

A responsive portfolio and case-study site built for GitHub Pages-compatible static hosting.

## Add another project

1. Add a project card to the `projects` array in `app/page.tsx`.
2. Create a new route at `app/projects/your-project/page.tsx`.
3. Reuse the case-study sections and styles from the Regional Utility Customer Portal page.
4. Replace the placeholder email address before publishing.

## Run locally

```bash
npm install
npm run dev
```

## Publish

Push the project to GitHub and connect the repository to a static-compatible host. The included project also supports the current deployment environment.
