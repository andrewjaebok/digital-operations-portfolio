# Digital Operations Portfolio

A responsive product operations portfolio hosted through GitHub Pages.

## Live portfolio

https://andrewjaebok.github.io/digital-operations-portfolio/

## Add another project

1. Add a project card to the `projects` array in `app/page.tsx`.
2. Create a new route at `app/projects/your-project/page.tsx`.
3. Reuse the case-study sections and styles from the Regional Utility Customer Portal page.
4. Replace the placeholder email address before publishing.

## Run locally

```bash
npm install
npm run dev
```

## Publish with GitHub Pages

Every push to `main` runs the GitHub Actions workflow and publishes the static site to GitHub Pages.
